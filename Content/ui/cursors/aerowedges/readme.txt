=== Aero Wedges Cursor Set ===

By: Haunty

Download: http://www.rw-designer.com/cursor-set/aerowedges

Author's decription:

Windows Vista Aero pointers without the tails. Pointer changes colors: green for link, blue for help, blinking red for working in background, rotating red pointer for busy.
